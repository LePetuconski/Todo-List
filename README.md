# To-do List  

To-do List feita em **Nuxt.js**, inicialmente desenvolvida para uso pessoal.

## 🚀 Começando

Essas instruções permitirão que você obtenha uma cópia do projeto em operação na sua máquina local para fins de desenvolvimento.

### 📋 Pré-requisitos

```
♦ Node.js
♦ NPM ou Yarn
```

### 🔧 Instalação

```
npm install
```

```
npm run dev
```
